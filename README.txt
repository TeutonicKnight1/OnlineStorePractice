Создание SPA приложения (интернет-магазин)

Одностраничное приложение созданое с помощью фраемворка React. Имеет 3 описанных в техническом задании страницы - cart,order и main(\).
Проект развернут без использования create-react-app.

Запуск проекта: Проект залит на бесплатный хостинг netlify по ссылке https://dynamic-granita-e7e8cd.netlify.app 
Для запуска локально небходимо установить необходимые модули командой "npm install", так как node-modules не добавлен в репозиторий
гитхаба. После установки запустить проект можно командой "npm start".  

Разворчаивание приложения без использования create-react-app:
    npm init -y (Создаст package.json)
    npm install react react-dom 
    npm install react-scripts
    package.json: "start": "react-scripts start"
